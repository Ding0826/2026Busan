## ✨ 功能列表

### 🏠 首頁 (Dashboard)
- **出發倒數計時**：自動計算距離 4/29 出發的天數
- **今日行程摘要**：根據當前日期自動顯示對應行程
- **花費統計**：總花費（韓元 + 台幣）即時顯示
- **今日花費進度條**
- **快速入口按鈕**：6 個快速跳轉按鈕
- **航班資訊卡片**：去程 / 回程航班高亮顯示

### 📅 行程表 (Plan)
- **卡片 + 時間軸**設計，呈現五天行程

  - 🍽 午餐 / 晚餐標籤清楚區分
  - ✈️ 航班時間特別高亮顯示
  - 🏨 住宿資訊標籤
  - ✍️ 日記輸入框（LocalStorage 即時保存）
  - 📋 一鍵複製當日行程按鈕

### 💰 記帳 (Wallet)

#### 💱 匯率換算工具
- **自訂匯率**：可輸入 1 台幣 = 幾韓元
- **支援算式輸入**：例如 `500 + 200 * 3`
- 即時換算韓元 ↔ 台幣

#### 📋 記帳功能
- **日期下拉選單**（固定為旅遊 5 天：4/29–5/3）
- **類別選擇**：餐飲 🍽 / 購物 🛍 / 交通 🚌 / 其他 📦
- **品項名稱** 輸入
- **金額**（韓元）輸入
- **收據照片**：
  - 支援拍照（調用相機）
  - 支援從相簿上傳
  - Canvas 壓縮至 400px → Base64 存 LocalStorage
  - 顯示縮圖，點擊放大查看

#### 📊 圖表統計（Chart.js）
- **長條圖**：每日花費對比（Day1–Day5）
- **圓餅圖（甜甜圈）**：各類別花費比例

#### 📄 記帳明細
- 每筆顯示：類別標籤 / 日期 / 品項 / 韓元 + 台幣換算
- 支援單筆刪除
- 支援一鍵清除全部

## 🚀 使用方法

### 方法一：直接開啟
1. 用瀏覽器開啟 `index.html`
2. 建議使用 Chrome / Safari（最新版）

### 方法二：手機使用（推薦）
1. 將 `index.html` 上傳至任何靜態網頁主機：
   - [GitHub Pages](https://pages.github.com/)
   - [Netlify Drop](https://app.netlify.com/drop)
   - [Vercel](https://vercel.com/)
2. 用手機瀏覽器開啟網址
3. 點「加入主畫面」儲存為類似 APP 的捷徑

### 方法三：區域網路分享
```bash
# 如已安裝 Python
python -m http.server 8080

# 或使用 Node.js serve
npx serve .
```
然後同一 WiFi 下的手機輸入電腦 IP 即可存取。

## 💾 資料儲存說明

所有資料儲存於瀏覽器 **LocalStorage**，不會上傳到任何伺服器。

| Key | 說明 |
|-----|------|
| `diary_day1` ~ `diary_day5` | 每日旅遊日記 |
| `expenses` | 記帳資料陣列（含圖片 Base64）|
| `check_carryon` | 隨身行李勾選狀態 |
| `check_checked` | 托運行李勾選狀態 |
| `memo` | 備忘錄文字 |

> ⚠️ 注意：清除瀏覽器快取或使用無痕模式會導致資料遺失，建議定期截圖備份重要記帳資料。
